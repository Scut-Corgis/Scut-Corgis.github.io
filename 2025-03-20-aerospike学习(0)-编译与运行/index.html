问题3. 编译absl失败2

fix:
step1. 升级g++版本

  1. yum 安装g++ 9
    sudo yum install devtoolset-9-gcc devtoolset-9-gcc-c++

  2. 指定新安装为默认版本
    在/etc/bashrc中增加：
    source /opt/rh/devtoolset-9/enable
